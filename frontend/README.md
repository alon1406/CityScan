# CityScan (Frontend)

CityScan is a modern, GIS-based platform designed to bridge the gap between residents and urban management. By leveraging real-time geolocation and interactive mapping, the platform enables communities to identify, report, and track infrastructure hazards efficiently.

This app is built with React, TypeScript, and Vite. Run from the `frontend` folder:

- `npm run dev` — start dev server
- `npm run build` — build for production
- `npm run preview` — preview production build
